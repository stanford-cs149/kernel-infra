43def _make_gpu_lookup(runner_map: dict[str, Type[Enum]]): 

44 lookup = {} 

45 for runner, gpus in runner_map.items(): 

46 for name, member in gpus.__members__.items(): 

47 if name.lower() in lookup: 

48 raise ValueError(f"Duplicate gpu name '{name}' found across Enums.") 

49 lookup[name.lower()] = GPU(name=name, value=member.value, runner=runner) 

50 return lookup

81class SubmissionMode(Enum): 

82 """ 

83 Different types of submission that can be made: 

84 Test: Run tests and give detailed results about passed/failed tests. These have short timeouts. 

85 Benchmark: Run larger benchmarks. Each benchmark is tested once, and then run multiple times. 

86 Profile: Gather profiling information. One selected benchmark is run under the profiler. No 

87 testing is performed in this mode (sometimes, you need to profile deliberately broken code) 

88 Leaderboard: Official submission to the leaderboard. This first runs public tests, then a 

89 repeated invocation of a single benchmark. Feedback for the secret benchmark is only very 

90 limited (no stdout/stderr). 

91 Private: Special run that does test followed by leaderboard (on a secret seed), but gives only 

92 very limited feedback. 

93 """
